How can I start learning Web Development ?

As said, there are plenty of resources available online for you to learn web development, but be careful of expensive bootcamps, some are worth it, while others are not.

If you are new to coding, then starting with the front-end of web development is the perfect way to ease yourself into the industry.

Frontend developers need to have a better knowledge of HTML, CSS, and JavaScript.

HTML is there to make the construction of the pages and screens. You are defining the content of the pages.
With CSS, you are describing how the HTML elements should be displayed.
In the end, you will use JavaScript to determine how the page and its elements should behave.

Both web developers and designers need to know these languages.

This makes front-end development the best place to start.
