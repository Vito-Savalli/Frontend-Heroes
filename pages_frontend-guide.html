The Ultimate 2023 Frontend Developer Roadmap

There is a ton of content on the web to teach you programming in 2023

From begginers guide to advanced tutorial, you will be able to a lot of ressources online.

But you may wonder, what do I need to learn ? How should I start ?

To help you answer this question, we built this step by step guide.

Get started to dive into the amazing world of frontend web developmment

1. Understand How the Web Works

Before diving into anything related to development, you need to understand How the Web Works.

You will learn what is an IP address, how computers communicate to each others via HTTP request and the client-server model.

How the Web Works: Free 4H Course

2. Learn Fundamentals of Computer Science

Now that you have learned how the web works, it is time to start learning basics of Computer Science.

Introduction to Computer Science will teach you how to think like a developer.

You will learn how to solve problems efficiently with diving into multiple programation languages.

Dont worry if you do not understand the languages used here !
The goal is to start thinking like a programmer

Introduction to Computer Science: a Free course published by Harvard.

3. Learn HTML and CSS

You learned how the web works and fundamentals of computer science ?

Well, it is time for you to actually start programming !

You should start to consider using a code editor in order to write your first lines of code.

Learn HTML along with CSS and start to build your first web page.

Visual Studio Code: The best code editor for Free

4. Start Building Small Components

Now that you have your code editor installed, you need to practice !

Start by building and styling small independants components.

Experiment with positionning, margin, and basics CSS style.

At this point you should be abble to implement and style buttons, images, text, and forms element.

Practice your code and solve challenges on Frontend Mentor

5. Build your first Single Page Website

Congrats ! you have made good progress already.

You should now consider building your first single page website.

You may want to understand the importance of Semantic HTML, SEO, metadata and how to host your website on Internet.

SEO for Beginners - Free 10H Course

6. Learn Responsive Design & Browser Accessibility

in 2022, 59.6% of all website traffic comes from people using mobile devices.

You want to be able to build components and layout that adapt to user screen size.

This is called Responsive Design.

However Responsive Design alone is not enough, you want you code to display according to your wish not only on your computer but on all browsers.

Learn about Browser Accessibility,
visit caniuse.com to see what CSS properties are supported by what browsers.

Responsive Web Design 300H Free Course on freecodecamp.org

7. Start Building your Portfolio

With all the knowledge you have acquired so far, you are now well armed to build your Portfolio.

Building your Portfolio is a great step into employment.

You want to show your future employer all the progress you have made so far and how you are willing to learn and apply what you have learned.

Check out our full guide - Build your Frontend Portfolio

8. Learn how to use Git & GitHub

Git is the world's most popular version control system (VCS).

Knowing how to use Git has become a mandatory skill in the world of coding.

There is plenty of ressources online to learn GitHub but here is the best tutorial for beginners.

Complete Beginner Tutorial - How to use GitHub

9. Dive Into JavaScript

Time to step your game to the next level !

JavaScript is used to create dynamic and interactiv content like applications and browsers.

At this point, if you want get the most of your time, you could go for a paid course but it is totally up to you.

With all the ressources in Internet you could learn it for free,
It will just takes more time and dedication.

Here is an amazing course I also took when I start programming two years ago.

you could easily buy it for just 11,99€

The Complete JavaScript Course 2023: From Zero to Expert

10. Understand Data Structures and Algorythms

Wait ? You did not have a headache already ?

Keep the hard work, there is 100% chances that you work will pay of but you must not give up now !

Data Structures and Algorythms are the fundations of web development.

You want to be able to understand how datas are represented and stored in the world of programming and what are Algortyhms.

Data Structures and Algorythms Free Certification Course

11. Learn about APIs

As a front-end developer, you will have to find a way to communicate with the guys in the backend room, to exchange data or verify a user.

To do so, you will probably use some APIs and HTTP.

What is an APIs and how do I send and receive data through it ?

Here is a 2H tutorial on YouTube that will answer the question !

APIs for Beginners - How to use an API

12. Practice your JavaScript

Now that you have learned about Javascript, Data Structures, Algorythms and APIs, you are not far away to build a real world looking website.

But learning is nothing without practicing.

Start by trying to build interactiv independants components such as advanced hover effect, pages animation or diaporama.

When you feel more confortbale, try to build an entire website with the use of JavaScript, HTML and CSS.

At this point you could consider to add JavaScript logic to your portfolio or use an APIs.

Remenber : Learning is nothing without practicing !

Learn JavaScript by solving coding exercises

13. Master Advanced JavaScript Concepts

In order to be proficient in JavaScript, you want to master those concepts :

Object Oriented Programming

Asynchronous JavaScript

Closures and Variable Scope

Classes and Prototypal Inheritance

Master all of this topics and get ready for the next step !

15 JavaScript concepts you should know

14. Start Learning a JavaScript Framework

Now that you feel confortable with JavaScript, you should definitively start to learn one of his framework.

Choose one these framework :

React.js

Vue.js

Angular.js

Start learning and try to build evertyhing you have done so far with this framework.

The average salary for a React Developer in Europe is 55000€

Average Developer Salary in Europe in 2022

Congrats for making it so far !

You should now have enough skills to work in a nice company.

Let me tell you one thing, she is pretty damn right !

You should always try to push yourself further and master new web technologies.
